**💰Expense Tracker**

A simple, full-stack expense tracking web app built with Spring Boot and Thymeleaf — track your income and expenses, view your balance in real time, and keep a complete transaction history, all tied to your own personal account.


**📖 Project Description**

Expense Tracker is a simple personal finance application where users can log in, add income and expense transactions, and view their current balance. Each user's data is stored separately, ensuring privacy. All transactions are saved in a MySQL database, so the data remains available even after restarting the application.

 This project was built using **Spring Boot** to learn core concepts such as **REST APIs, Thymeleaf, session management, Spring Data JPA, and MySQL database integration**.

**Features**


🔐 Simple Login – Sign in using just your name and email. No password required.

👤 Private User Accounts – Each user can only view and manage their own transactions.

💰 Balance Dashboard – Instantly see your current balance, total income, and total expenses.

➕ Add Transactions – Easily record income or expenses in a single form. Positive amounts are treated as income, while negative amounts are treated as expenses.


📋 Transaction History – View all your past transactions in one place, with green highlighting income and red highlighting expenses for quick identification.


💾 Persistent Storage – All your transaction data is securely stored in a MySQL database and remains available even after restarting the application.

**🔮 Future Enhancements**


🔑 Real authentication with hashed passwords

✏️ Edit and delete existing transactions

🏷️ Categorize transactions (food, rent, salary, etc.)

📅 Filter history by date range
